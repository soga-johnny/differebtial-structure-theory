# 概念統合プロトコル

> **v0.5.0** | [← README](../README.md)

---

## 1. 原則

### Straはアナロジー型思想体系である

Straは既存の哲学・思想・宗教・社会理論から構造的に類似した概念を借用し、固有の文脈に再適用することで構成される。これを「アナロジー型統合」と呼ぶ。

アナロジー型統合において以下の点を原則とする：

- 概念の起源を常に明示する
- 借用は常に部分的であり、Straとの差分（=Stra固有の定義・目的・文脈）が必ず発生する
- その差分を明示的に定義することが、Straの概念を明確にする
- 起源との関係性を分類・透明化することで知的誠実性を担保する

> Straが「差分を記述することで意味を定義する」という方法論を採る以上、  
> 他概念との差分の定義もまた、Straの自己定義として機能する。

---

## 2. 関係性の分類

他思想体系とStraの概念の関係性は以下5種に分類する。

| 種別 | 定義 | 例 |
|---|---|---|
| **継承（Derived）** | 元概念の定義をほぼ維持した上でStraの文脈に適用する | Bourdieuの資本3分類 → Straの資本種別 |
| **並列（Parallel）** | 独立して生まれた類似構造。直接の系譜はないが相互参照可能 | ベイズ推定 ↔ Straの中央値動的補正 |
| **拡張（Extended）** | 元概念の適用範囲をStraが新しい領域に延伸する | Lévi-Strauss構造主義 → 個人のアイデンティティ記述への適用 |
| **部分借用（Partial）** | 概念の一部を採用し、残部はStraが独自定義する | Deleuzの差異の哲学 → 「差分が意味の源泉」という命題のみ採用 |
| **対照（Contrast）** | 同一の問いに対する異なる回答として並置する | 本質主義 vs Stra（非本質主義的差分論） |

---

## 3. 統合プロトコル

既存思想体系からの概念をStraに統合する際は以下の手順を経る。

### Step 1: 起源の明示

```
起源思想体系: [思想・宗教・理論の名称]
起源概念名:   [元の概念名（可能であれば原語）]
参照文献:     [著作・文書名]（任意）
```

### Step 2: 元概念の定義

起源思想体系における元概念の定義を、その文脈から切り離さずに記述する。  
（Straの文脈で解釈し直した定義ではなく、元の定義を忠実に記述する）

### Step 3: Straでの使用定義

Straにおいてその概念をどのように使用・適用するかを定義する。

### Step 4: 差分の定義

|  | 元概念 | Stra |
|---|---|---|
| 目的 | | |
| 適用対象 | | |
| 前提 | | |
| 方法 | | |

差分が存在しない場合（完全一致）は継承として分類し、その旨を明記する。  
差分が存在する場合は各軸で具体的に定義する。

### Step 5: 関係性の分類

Step 2 で定義した分類種別（継承・並列・拡張・部分借用・対照）を決定し、記録する。

### Step 6: 統合箇所の記録

Straのどのドキュメントのどの概念として統合したかを記録する。

## 5. 既存統合の記録

以下は現時点でStraに統合済みの概念の記録である。

---

### 資本の3分類 ← Bourdieu（資本理論）

**起源概念の定義**  
ブルデューは資本を経済資本（金銭・財産）・社会関係資本（人脈・ネットワーク）・文化資本（知識・技能・学歴・文化的嗜好）の3種に分類した。資本は社会的場（champ）において機能し、場ごとに変換可能とした。

**Straでの使用定義**  
差分が評価圏において認識・評価されたとき変換される先の資本の種別として使用する。差分の種類と参入する評価圏の組み合わせによって、どの種類の資本に変換されるかが決まる。

**差分の定義**

|  | Bourdieu | Stra |
|---|---|---|
| 目的 | 社会構造・階級再生産の記述 | 個人・組織の差分設計と資本化の実践支援 |
| 起点 | 生まれ持つ資本の不平等な分布 | 差分の設計・投資・評価圏への参入 |
| 前提 | 場の支配者による不平等な評価基準 | 評価圏の選択と入力差分の設計が主体に開かれている |
| 方法 | 構造分析・社会批評 | 差分設計のプロセスとして実装 |

**関係性の種別**: 部分借用  
**統合先**: `sphere/03-evaluation-capital.md § 3`

---

### 構造主義 ← Lévi-Strauss

**起源概念の定義**  
意味は対象の本質に内在するのではなく、差異の体系（structure）の中での位置関係から生まれる。神話・親族構造・言語など、表面上異なるものも深層構造は二項対立の組み合わせとして記述できる。

**Straでの使用定義**  
「意味は差分の関係から生まれる」という構造的立場をStraの存在論的出発点として採用する。所属・中央値・差分の体系はこの構造を継承している。

**差分の定義**

|  | 構造主義 | Stra |
|---|---|---|
| 目的 | 文化・神話・言語の深層構造の記述 | 個人・組織・プロダクトのアイデンティティ記述と設計 |
| 適用対象 | 社会・文化・言語現象 | 個人・組織・プロダクト・ブランドなど主体全般 |
| 前提 | 二項対立的深層構造 | 多軸の差分ベクトル（二項対立に限定しない） |
| 方法 | 構造分析（記述） | 差分の測定・設計・資本変換（実装まで） |

**関係性の種別**: 拡張  
**統合先**: `sphere/01-difference-axioms.md § 4-1` / `sphere/05-references.md § 1`

---

### 縁起（pratītyasamutpāda） ← 仏教

**起源概念の定義**  
仏教における縁起とは「すべての現象は相互依存的な関係性によって生起する」という命題である（「此あれば彼あり、此なければ彼なし」）。固定した本質を持つ独立した存在はなく、関係性の総体として存在する。苦（dukkha）からの解脱を目的とした実践体系の認識論的基盤として機能する。

**Straでの使用定義**  
「存在は関係性から定義される」という命題において縁起と並列する。Straにおける「アイデンティティ = 複数所属における差分の総体」という定式はこの関係論的存在観と構造的に近い。

**差分の定義**

|  | 縁起 | Stra |
|---|---|---|
| 目的 | 苦からの解脱（涅槃への道） | 豊かさの実現（5条件の同時成立） |
| 存在の記述 | 相互依存による生起・消滅 | 参照基準（中央値）からの差分の測定と設計 |
| 規範的方向 | 執着の解消・無我の体得 | 差分の意図的な設計・資本化 |
| 実践 | 修行・観察・手放し | 入力差分の設計・評価圏への参入・補正ループ |

**関係性の種別**: 並列  
**統合先**: 未統合（参照理論への追記候補）

---

### 無我（anattā / 非実体性） ← 仏教

**起源概念の定義**  
「固定した・不変の・独立した自我（ātman）は存在しない」という仏教の中心的命題。自己とは五蘊（色受想行識）の仮の集合であり、固有の本質（自性）はないとする。

**Straでの使用定義**  
Straの非本質主義（アイデンティティは固定本質ではなく差分の総体）と並列する。ただし目的と方法論は大きく異なる。

**差分の定義**

|  | 無我 | Stra |
|---|---|---|
| 目的 | 自我への執着の解消による苦からの解脱 | 差分の記述・設計を通じた豊かさの実現 |
| 結論 | 自己は幻であり手放すべき | 自己（= 差分の総体）は設計・運用するべき |
| 実践 | 瞑想・観察・解体 | 所属の認識・差分の測定・入力差分の設計 |
| 価値観 | 欲望・執着からの自由 | 選択肢・意志・実行基盤の確保（豊かさ） |

**関係性の種別**: 対照（同じ「固定本質の否定」から正反対の実践へ）  
**統合先**: 未統合（批評・対照概念として参照理論への追記候補）

---

### 差異の哲学（différance） ← Deleuze / Derrida

**起源概念の定義**  
Deleuzにおける差異（difference-in-itself）は、同一性や対立によって定義される従属的差異ではなく、差異それ自体を存在論の第一原理とする。Derridaのdifféranceは意味が常に延期・分散されることを示す。

**Straでの使用定義**  
「差分が意味を構成する」という命題において並列・部分的に共鳴する。ただしStraは差分の前に参照基準（中央値）を置く点で立場が逆転する。

**差分の定義**

|  | 差異の哲学 | Stra |
|---|---|---|
| 差分の順序 | 差分が先にあり、基準はそこから生まれる | 中央値（基準）が先にあり、差分はその逸脱量 |
| 目的 | 同一性・固定性の解体 | 差分の測定・設計・資本化 |
| 適用 | 形而上学・存在論 | 個人・組織・プロダクトの実践的設計 |

**関係性の種別**: 対照  
**統合先**: `sphere/05-references.md § 2`
